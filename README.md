# AirDraw AI: Gesture-Based Virtual Drawing Application

AirDraw AI is a real-time computer vision project that allows users to draw on the screen using only hand gestures.  
The application uses a webcam to detect hand landmarks and lets the user draw virtually in the air using the index finger.

## Project Demo

Users can draw, erase, clear, change colors, and save drawings using hand gestures and keyboard controls.

## Features

- Real-time hand tracking
- Virtual air drawing using index finger
- Color selection options
- Eraser mode
- Clear screen option
- Save drawing as image
- Webcam-based gesture control
- Built using Python, OpenCV, and MediaPipe

## Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy
